Manual QA Project - Webshop Testing

Description
This project demonstrates manual testing of a webshop application.  
The focus is on validating core functionalities and identifying defects.

Scope
The following features were tested:
- User login
- Product search
- Add to cart
- Cart management

Test Activities
- Writing detailed test cases
- Executing manual tests
- Identifying and documenting bugs

Deliverables
- Test cases (test-cases.md)
- Bug reports (bug-reports.md)

Tools & Skills
- Manual testing
- Test case design
- Bug reporting
- Basic SQL knowledge
- (optional: Postman for API testing)

Goal
The goal of this project is to demonstrate practical QA skills and testing approach in a real-world scenario.
